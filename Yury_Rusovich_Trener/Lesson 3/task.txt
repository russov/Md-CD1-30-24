Задание: Работа с файлами и циклами

1. Напишите программу, которая запрашивает у пользователя набор целых чисел с клавиатуры и сохраняет их в файл "numbers.txt", каждое число на отдельной строке.
2. Добавьте функцию, которая считывает числа из файла "numbers.txt" и выводит их на экран в обратном порядке.
3. Расширьте программу, чтобы она считывала числа из файла "numbers.txt" и находила их сумму. Сумма должна выводиться на экран.
4. Напишите функцию, которая считывает числа из файла "numbers.txt" и находит среднее арифметическое всех чисел. Результат должен быть выведен на экран.
5. Добавьте возможность дописывать новые числа в конец файла "numbers.txt", а не перезаписывать его полностью.

Дополнительное задание:

6. Реализуйте проверку ввода пользователя на корректность (например, чтобы программа не ломалась при вводе нечисловых данных).
